Application Overview

What problems does this application attempt to solve?

Problems with Training Presentations

  • Interactivity
  • Data Collection
  • Ellimination of Redundancy
  • Mobility

Solutions

  • Interactivity: A personal UI for customers
  • Data Collection: Have that UI send meaningful data to a database
  • Ellimination of Redundacy: Have customers vote on what material with a yes or no
  • Mobility: Make mobile sites and web based presentations

Customer User Interface Overview

There are five sections to the interface:
  • Polling/Voting
  • Request Time
  • Evaluation
  • Links
  • Messaging

Polling/Voting

Allows the users to vote or answer questions about a section.

Request Time

A subtle way to ask for more time on a slide

Evaluation

Real time evaluations for any section of material.
Content Effectivess: Is the content clear and relevant?
Trainer's Effectiveness: Is the trainer engaging and presenting the material good?

Links

Links relevant to the slide or section

Messaging

Send Messages to the instructor either anonymously or send them with your name to get answered later
